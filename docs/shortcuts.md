# Puppet Master 단축키

macOS / Windows(Linux) 기준 단축키 목록. macOS는 `Cmd`, Windows는 `Ctrl`을 사용하며 나머지 키는 동일하다.

## 파일

| 기능 | macOS | Windows |
| --- | --- | --- |
| 폴더 열기 (프로젝트) | `Cmd+O` | `Ctrl+O` |
| 저장 | `Cmd+S` | `Ctrl+S` |
| 창 닫기 / 종료 | `Cmd+W` (창 닫기) | `Alt+F4` (종료) |

최근 프로젝트는 **File → Open Recent** 메뉴에서 열 수 있다 (단축키 없음).

## 탐색 (정의 점프 / 히스토리)

| 기능 | macOS | Windows |
| --- | --- | --- |
| 정의로 이동 (커서 위치 심볼) | `F12` | `F12` |
| 정의로 이동 (클릭) | `Cmd+클릭` | `Ctrl+클릭` |
| 뒤로 가기 | `Opt+←` | `Alt+←` |
| 앞으로 가기 | `Opt+→` | `Alt+→` |
| 뒤로 가기 (에디터/입력란 밖에서만) | `Backspace` | `Backspace` |
| 뒤로 / 앞으로 | 마우스 뒤로/앞으로 버튼 (4번/5번 버튼) | 동일 |

## 검색

| 기능 | macOS | Windows |
| --- | --- | --- |
| 통합 검색 열기/닫기 (심볼 + 전문 검색) | `Cmd+Shift+F` | `Ctrl+Shift+F` |
| 검색 결과 이동 | `↑` / `↓` | `↑` / `↓` |
| 선택한 결과로 이동 | `Enter` | `Enter` |
| 검색 닫기 | `Esc` | `Esc` |

에디터 내부 검색은 Monaco 기본 단축키를 따른다 (`Cmd+F` / `Ctrl+F`).

## 편집 / 리팩터링

| 기능 | macOS | Windows |
| --- | --- | --- |
| Smart Rename (커서 위치 심볼) | `F2` | `F2` |
| Rename 미리보기에서 적용 | `Enter` (새 이름 입력란) | `Enter` |
| Rename 취소 | `Esc` | `Esc` |

## 북마크

| 기능 | macOS | Windows |
| --- | --- | --- |
| 북마크 토글 (현재 줄) | `Cmd+F2` | `Ctrl+F2` |

북마크 목록은 좌측 패널 Bookmarks 섹션에서 클릭으로 이동한다.

## AI 자동완성 (고스트 텍스트)

| 기능 | macOS | Windows |
| --- | --- | --- |
| 설정 열기/닫기 (provider·모델·API 키) | `Cmd+,` | `Ctrl+,` |
| 제안 수락 | `Tab` | `Tab` |
| 제안 무시 | `Esc` 또는 계속 입력 | `Esc` 또는 계속 입력 |

## 보기 (창 전체 줌 — 에디터 폰트 포함)

| 기능 | macOS | Windows |
| --- | --- | --- |
| 확대 | `Cmd+=` (`Cmd+Plus`) | `Ctrl+=` (`Ctrl+Plus`) |
| 축소 | `Cmd+-` | `Ctrl+-` |
| 원래 크기 | `Cmd+0` | `Ctrl+0` |

Electron 표준 View 메뉴 기능으로, UI 전체가 함께 확대/축소된다.

## 오버레이 공통

| 기능 | macOS | Windows |
| --- | --- | --- |
| 오버레이 닫기 (검색 / Rename / 설정) | `Esc` 또는 배경 클릭 | `Esc` 또는 배경 클릭 |
